### data type
1) numbers
2) Bigints
3) strings
4) booleans
    - 참과 거짓 판별
5) null
6) undefined
    - 비어있는 string문 (ex) '' , 0 등등
7) symbols
8) objects
    - properties(속성)을 가짐. 도트 연산자를 통해 쓸 수 있음. (ex) 'Happy'.length -> 문자열의 길이를 알려줌.
    - methods를 가짐. (ex) ' space '.trim() -> 도트 연산자를 사용하며, 공백을 없애줌.
### 연산자(operater)
1) 산술 연산자

    - +=,++,--,-+
    - === 비교 연산자 (같은지 판별) vs = 대입 연산자
    - >=, <=
2) 논리 연산자
    - &&(and), ||(or)
